# AI Kundtjänst

En avancerad AI-baserad kundtjänst byggd med Node.js, lik stora AI-chattar.

## Funktioner
- Chatt med AI för olika företag (law, tech, cleaning).
- Sessionshistorik (sparas lokalt).
- Feedback-knappar.
- Rensa chatt.
- Mörk/ljus tema.
- Responsiv design.

## Starta projektet
```bash
npm install
npm start
```
Öppna `http://localhost:3000?company=law` i webbläsaren.
